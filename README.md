# 🎓 Full-Stack E-Learning Platform (Collaborative Project)

This project is a modern E-Learning solution built as a final-year project for a senior colleague. I participated as a **Technical Contributor**, focusing on accelerating the development process through prompt engineering and UI implementation.

## 🚀 My Contributions
* **UI/UX Implementation:** Assisted in building responsive components using **Tailwind CSS** and **Framer Motion**.
* **Feature Acceleration:** Leveraged AI prompting to generate boilerplate code for server routes and frontend structures.
* **Learning Outcome:** Gained deep insights into Full-Stack architecture, Vite-based builds, and server-side logic management.

## 🛠️ Tech Stack (Full-Stack)
* **Frontend:** React.js (Vite), Tailwind CSS, Lucide React
* **Animations:** Framer Motion
* **Backend:** Node.js, Express
* **Database:** Better-SQLite3
* **AI Integration:** Google Generative AI SDK (@google/genai)

## 📁 Project Highlights
* **Vite-powered** fast development environment.
* **AI-integrated** features for enhanced learning.
* **Modular Backend** handling user authentication and course data.
